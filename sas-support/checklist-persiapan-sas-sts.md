# Checklist Persiapan SAS / STS

## 1. Sebelum Ujian
- [ ] Cetak LJK sesuai template.
- [ ] Pastikan kotak sensor atas (3) dan bawah (2) terlihat jelas.
- [ ] Siapkan database peserta CSV.
- [ ] Siapkan kunci jawaban final.

## 2. Saat Scan
- [ ] Upload template pada aplikasi.
- [ ] Geser sensor TOP1-TOP3 dan BOT1-BOT2 sesuai template.
- [ ] Geser area nomor peserta dan area jawaban.
- [ ] Uji 1 lembar sebelum batch penuh.

## 3. Setelah Scan
- [ ] Cek nomor peserta yang tidak terbaca.
- [ ] Koreksi manual jika perlu.
- [ ] Export Excel hasil.
- [ ] Download backup JSON untuk arsip dan lanjut pengerjaan.

## 4. Dokumentasi Sekolah
- [ ] Simpan file Excel final.
- [ ] Simpan backup JSON final.
- [ ] Simpan template database + kunci jawaban.
